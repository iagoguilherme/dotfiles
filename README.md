# Dotfiles

Minhas configurações de terminal pra macOS.

## Stack

- **Ghostty** — emulador de terminal
- **JetBrains Mono Nerd Font** — fonte com ícones
- **eza** — substituto moderno do `ls`
- **Tema Darkside** — visual escuro

## Instalação em um Mac novo

```bash
git clone https://github.com/iagoguilherme/dotfiles.git ~/dotfiles
cd ~/dotfiles
./install.sh
```

## Atalhos do Ghostty

- `Cmd + Shift + Espaço` — quick terminal global (em qualquer app)
- `Cmd + T` — nova aba
- `Cmd + D` — split horizontal
- `Cmd + Shift + ,` — recarregar config

## Aliases do shell

- `ls` — listagem com ícones
- `ll` — detalhada com git status
- `la` — inclui arquivos ocultos
- `lt` — árvore de 2 níveis
